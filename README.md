# NewsFeed

A Simple Android Mobile Application which has been implemented using MVVM Clean Architecture.
Here are the technology stacks are being used mentioned below

## Tech Stack

- Clean Architecture
- MVVM Pattern
- Kotlin
- Coroutines, Flow
- Dagger Hilt
- Retrofit2
- Navigation Architecture Component
- Unit tests
